# Robótica - Publisher/Subscriber

Exemplos de comunicação Publisher/Subscriber em ROS.

## Descrição

Implementações de padrão Publisher/Subscriber usando ROS (Robot Operating System) em C++ e Python.

## Conceitos

- ROS Topics
- Publisher/Subscriber Pattern
- Comunicação entre Nós ROS
- Message Passing

## Arquivos

- `comunicacao.cpp` - Implementação de comunicação em C++
- `comunicacao.py` - Implementação de comunicação em Python
- `escutador.cpp` - Subscriber em C++
- `escutador.py` - Subscriber em Python
- `teste.cpp` - Testes em C++
- `teste.py` - Testes em Python

## Tecnologias

- ROS (Robot Operating System)
- C++
- Python
